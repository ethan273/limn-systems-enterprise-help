# Admin & System Management

System administration, user management, and integration monitoring.

## Overview

The admin system provides comprehensive tools for managing users, roles, approvals, portals, APIs, security, and system settings.

## Key Modules

### User & Access Management
- **User Management** - Manage user accounts
- **Role Management** - Configure roles and permissions
- **Approvals** - Approve user requests
- **Portal Management** - Manage portal access

### API & Integration Management
- **API Management** - Manage API keys and access
- **API Key Rotation** - Rotate API keys securely
- **API Health** - Monitor API health and performance
- **API Security** - API security settings
- **Integration Status** - Monitor integration health

### Security & Monitoring
- **Security Events** - Security event monitoring
- **Activity Logs** - System activity logs
- **System Settings** - Global system configuration
- **Data Export** - Export system data

## Key Features

- User lifecycle management
- Role-based access control
- Approval workflows
- API management
- Security monitoring
- Activity logging
- Integration health monitoring
- Data export tools
