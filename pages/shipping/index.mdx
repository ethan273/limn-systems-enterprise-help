# Shipping Management

Manage shipments, tracking, carriers, and deliveries.

## Overview

The shipping management system provides tools for creating shipments, tracking deliveries, managing carriers, and coordinating pickups. Integrates with SEKO Logistics for real-time tracking.

## Key Modules

- **Shipments** - Create and manage shipments
- **Tracking Dashboard** - Real-time shipment tracking
- **Carriers** - Carrier management and selection
- **SEKO Integration** - SEKO Logistics integration
- **Pickup Requests** - Schedule carrier pickups
- **Delivery** - Delivery coordination and confirmation

## Key Features

- Multi-carrier support
- Real-time tracking
- SEKO Logistics integration
- Pickup scheduling
- Delivery notifications
- POD (Proof of Delivery)
- Integration with production
