# Global Features

System-wide features available throughout the application.

## Overview

Global features are available across all modules and provide core functionality for search, notifications, messaging, settings, and more.

## Key Features

- **Global Search** - Search across all entities and documents
- **Notifications** - In-app notification system
- **Messaging** - Internal messaging and communication
- **User Settings** - Personal preferences and settings
- **Theme** - Light/dark mode and visual customization
- **Offline Mode** - Work offline with sync when online
- **Storage** - File storage and management

## Search Capabilities

The global search feature allows you to quickly find:
- Orders and projects
- Customers and contacts
- Products and materials
- Documents and files
- Tasks and workflows

## Notifications

Receive real-time notifications for:
- Task assignments
- Order updates
- Approval requests
- System alerts
- Messages and mentions
