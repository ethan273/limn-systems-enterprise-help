# Production Management

Track manufacturing from production orders through quality control and packing.

## Overview

The production management system provides comprehensive tools for managing manufacturing operations, including production orders, shop drawings, quality control, and packing. Integrates with factory partners and QC teams.

## Key Modules

- **Production Orders** - Manufacturing work orders
- **Production Tracking** - Real-time production status
- **Shop Drawings** - Technical drawings and specifications
- **Factory Reviews** - Factory approval workflows
- **Quality Control** - QC inspections and testing
- **QC Templates** - Reusable inspection templates
- **QC Mobile App** - Mobile inspection tool
- **Production Batches** - Batch tracking and management
- **Production Stages** - Stage-based workflow
- **Packing** - Packing list and preparation
- **Production Reset** - Reset and restart production
