# Dashboards

Limn Systems Enterprise provides 10 specialized dashboards for different roles and use cases.

## Available Dashboards

- **Main Dashboard** - Overview of key metrics and system status
- **Analytics Dashboard** - Data visualization and business insights
- **Executive Dashboard** - C-level KPIs and strategic metrics
- **Financial Dashboard** - Revenue, expenses, and profitability
- **Projects Dashboard** - Project health and progress tracking
- **Manufacturing Dashboard** - Production metrics and capacity
- **Design Dashboard** - Design team performance and deliverables
- **Shipping Dashboard** - Logistics tracking and delivery metrics
- **Quality Dashboard** - QC metrics and quality trends
- **Partners Dashboard** - Partner performance and collaboration

## Getting Started

Select a dashboard from the navigation menu to learn more about each dashboard's features and capabilities.

## Key Features

- Real-time data updates
- Customizable widgets and layouts
- Role-based access control
- Export and reporting capabilities
- Mobile-responsive design
