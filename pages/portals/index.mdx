# External Portals

Self-service portals for customers, designers, factories, and QC teams.

## Overview

The portal system provides secure, role-based access for external partners including customers, designers, factories, and QC teams. Each portal type has customized features and modules.

## Portal Types

- **Customer Portal** - Self-service for customers
- **Designer Portal** - Collaboration portal for designers
- **Factory Portal** - Production portal for manufacturers
- **QC Portal** - Quality control inspection portal

## Key Features

- Role-based portal access
- Customizable module access
- Document sharing
- Order and project visibility
- Quality reports and inspections
- Secure authentication
- Mobile-responsive design

## Portal Management

- **Portal Access** - Grant and manage portal access
- **Portal Modules** - Configure available modules per user
- **Invitations** - Send portal invitations to partners
