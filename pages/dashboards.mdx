## Common Features

All dashboards share these powerful capabilities:

### Auto-Refresh

**How It Works:**
- Data automatically refreshes every 60 seconds
- No need to reload the page
- Real-time metrics without manual intervention

**Manual Refresh:**
- Click the **Refresh** button (circular arrow icon) in the dashboard header
- Forces immediate data reload
- Useful after making changes in other parts of the system

### Date Range Filtering

**Available Ranges:**
- **7d** - Last 7 days
- **30d** - Last 30 days (default)
- **90d** - Last 90 days
- **1y** - Last 12 months
- **All** - All historical data

**How to Change:**
1. Locate the date range selector in the dashboard header
2. Click the current range (e.g., "30d")
3. Select your desired time period
4. Dashboard updates automatically

**Important Notes:**
- Some metrics (like revenue trends) always show 12 months of data
- Date ranges apply to summary metrics and filters
- Changing ranges does not affect historical trend charts

### PDF Export

**Export Your Dashboard:**
1. Click the **Export PDF** button in the dashboard header
2. System generates a PDF snapshot of the current view
3. PDF includes:
   - Current date range
   - All visible metrics and charts
   - Dashboard title and metadata
4. Download begins automatically

**Best Practices:**
- Export at month/quarter end for records
- Include in executive reports
- Share with stakeholders who don't have system access

## Troubleshooting

### Dashboard Not Loading

**Symptoms:**
- Spinning loader indefinitely
- Error message displayed
- Blank screen

**Solutions:**

1. **Check Internet Connection**
   - Ensure you're online
   - Test other pages
   - Refresh browser

2. **Clear Browser Cache**
   - Press Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
   - Clear cache in browser settings
   - Try incognito/private mode

3. **Verify Permissions**
   - Ensure you have dashboard access
   - Contact admin if permission denied
   - Check your role permissions

4. **Try Different Browser**
   - Test in Chrome, Firefox, Safari, or Edge
   - Update browser to latest version
   - Disable browser extensions temporarily

5. **Contact Support**
   - If issue persists, report to IT
   - Include: Dashboard name, error message, browser version
   - Provide screenshot if possible

### Data Looks Incorrect

**Symptoms:**
- Numbers seem wrong
- Missing data points
- Outdated information

**Solutions:**

1. **Check Date Range**
   - Verify selected date range is correct
   - Try changing to different range
   - Ensure you're comparing correct periods

2. **Force Refresh**
   - Click refresh button in dashboard header
   - Wait for data to reload
   - Check if numbers update

3. **Verify Source Data**
   - Navigate to detail pages (Orders, Invoices, etc.)
   - Manually verify counts
   - Ensure data entry is complete

4. **Check Filters**
   - Some dashboards may have additional filters
   - Reset filters to default
   - Try "All" date range

5. **Report Issue**
   - If data is genuinely incorrect, report to support
   - Include: Dashboard name, metric name, expected vs actual
   - Provide date range used

### Charts Not Displaying

**Symptoms:**
- Empty chart areas
- "No data available" message
- Chart rendering issues

**Solutions:**

1. **Expand Date Range**
   - Try "All" date range
   - May not have data for selected period
   - New accounts may have limited data

2. **Check Browser Compatibility**
   - Update to latest browser version
   - Try different browser
   - Ensure JavaScript is enabled

3. **Zoom Out/In**
   - Browser zoom may affect chart rendering
   - Reset zoom to 100%
   - Refresh page

4. **Wait for Data**
   - New installations may not have historical data
   - Charts populate as data accumulates
   - Some metrics require minimum data points

### PDF Export Failing

**Symptoms:**
- Download doesn't start
- Incomplete PDF
- Export button not working

**Solutions:**

1. **Check Popup Blocker**
   - Allow popups from your domain
   - Temporarily disable blocker
   - Try export again

2. **Scroll to Top**
   - Ensure dashboard is scrolled to top
   - Export captures visible content
   - All sections should be loaded

3. **Reduce Date Range**
   - Very large datasets may cause timeout
   - Try shorter date range
   - Export multiple PDFs if needed

4. **Try Different Browser**
   - Some browsers handle PDF better
   - Chrome generally most reliable
   - Update browser to latest version

### Performance Issues

**Symptoms:**
- Dashboard loads slowly
- Charts lag when interacting
- Browser freezes

**Solutions:**

1. **Reduce Date Range**
   - Shorter ranges load faster
   - Start with 7d or 30d
   - Increase gradually if needed

2. **Close Other Tabs**
   - Dashboards are data-intensive
   - Free up browser memory
   - Restart browser if needed

3. **Check Internet Speed**
   - Run speed test
   - Use wired connection if possible
   - Contact ISP if consistently slow

4. **Optimize Computer**
   - Close unnecessary applications
   - Ensure adequate RAM
   - Update operating system

## FAQ

### General Questions

**Q: How often do dashboards update?**

A: Automatically every 60 seconds. You can also manually refresh by clicking the refresh button in the dashboard header.

**Q: Can I customize which dashboards I see?**

A: Dashboard visibility is based on your role permissions. Contact your administrator to request access to additional dashboards.

**Q: Why don't I see all 10 dashboards?**

A: Dashboard access is role-based. Executives see all dashboards, while team members see relevant ones for their function (e.g., designers see Design Dashboard, not Financial Dashboard).

**Q: Can I create custom dashboards?**

A: Currently, dashboards are pre-configured. Custom dashboard creation may be added in future releases. Use the Analytics Dashboard for comprehensive cross-functional metrics.

**Q: Do dashboards work offline?**

A: No, dashboards require an internet connection to load real-time data. However, you can export PDFs for offline viewing.

### Data & Metrics

**Q: What date range should I use?**

A:
- **Daily operations:** 7d or 30d
- **Monthly reviews:** 30d or 90d
- **Quarterly planning:** 90d or 1y
- **Historical analysis:** All

**Q: Why is my revenue different on different dashboards?**

A: Different dashboards may show different revenue calculations:
- **Executive:** Total revenue (all sources)
- **Financial:** Collected revenue (cash basis)
- **Analytics:** Order revenue (may include pending)

Always check the date range and metric definition.

**Q: How is "on-time" calculated?**

A:
- **Projects:** Completed by end_date
- **Deliveries:** Delivered by/before promised_date
- **Production:** Completed by scheduled_completion_date

**Q: What's the difference between "invoiced" and "revenue"?**

A:
- **Invoiced:** Amount billed (may not be paid yet)
- **Revenue:** Amount collected (cash received)
- **A/R (Accounts Receivable):** Invoiced but not yet paid

### Troubleshooting

**Q: Dashboard shows "No data available" - why?**

A: Possible reasons:
- Date range has no activity (try expanding range)
- New account with no historical data
- Permissions prevent seeing data
- Data filters excluding all results

**Q: Charts are empty but metrics show numbers - what's wrong?**

A: Usually a browser rendering issue:
1. Try different browser
2. Clear cache and refresh
3. Disable browser extensions
4. Update browser to latest version

**Q: Export to PDF shows partial data - how to fix?**

A:
1. Scroll to top of dashboard before exporting
2. Wait for all charts to fully load
3. Try shorter date range (less data = faster export)
4. Use Chrome for most reliable PDF generation

**Q: Dashboard performance is slow - how to improve?**

A:
- Use shorter date ranges (7d or 30d instead of All)
- Close other browser tabs
- Check internet connection speed
- Clear browser cache
- Restart browser

### Advanced Usage

**Q: Can I schedule automated dashboard reports?**

A: Automated reporting is planned for a future release. Currently, you can manually export PDFs and email them on your desired schedule.

**Q: Can I embed dashboards in other tools?**

A: Dashboard embedding is not currently supported. You can share screenshots or PDF exports.

**Q: How do I compare my performance to industry benchmarks?**

A: Benchmarking data is not built-in. You can export your metrics and compare manually to industry data from your sources.

**Q: Can I export raw data for custom analysis?**

A: Yes! Navigate to the detail pages (Orders, Projects, etc.) and use the export features to get CSV data for custom analysis in Excel or other tools.

**Q: How far back does historical data go?**

A: Historical data goes back to your account creation date or data migration date. There's no time limit on data retention.
