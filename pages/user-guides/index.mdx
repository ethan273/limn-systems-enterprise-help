# User Guides

Role-specific guides for different user types.

## Overview

Comprehensive guides tailored to different roles and use cases. Each guide provides step-by-step instructions for common tasks relevant to that role.

## Available Guides

- **Admin Guide** - System administrators and super admins
- **Designer Guide** - Internal and external designers
- **Customer Guide** - Customer portal users
- **Factory Guide** - Manufacturing partner users
- **QC Guide** - Quality control inspectors
- **Sales Guide** - Sales team members
- **Finance Guide** - Finance and accounting users
- **Production Guide** - Production managers and coordinators

## What's Included

Each user guide covers:
- Getting started with your role
- Common daily tasks
- Best practices
- Tips and shortcuts
- Role-specific features
- Workflow examples

## Getting Started

Select the guide that matches your role to learn how to use the platform effectively for your specific needs.
