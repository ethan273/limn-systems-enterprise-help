# Designer Guide

Complete guide for designers using the Limn Systems Designer Portal.

## Getting Started

### Accessing the Designer Portal
1. Navigate to app.limn.us.com/auth/designer
2. Enter your email address
3. Click "Send Magic Link"
4. Check your email and click the link
5. You'll be logged in automatically

### Portal Overview
The Designer Portal allows you to:
- View assigned design projects
- Submit design deliverables
- Access project documents
- View quality reports
- Collaborate with Limn team

## Dashboard

Your dashboard displays:
- **Active Projects** - Current assignments
- **Pending Submissions** - Deliverables awaiting upload
- **Recent Uploads** - Latest document submissions
- **Project Notifications** - Updates from Limn team
- **Quality Reports** - Inspection results

## Managing Projects

### Viewing Projects
1. Navigate to Projects
2. View all assigned projects
3. Filter by:
   - Status (active, pending approval, completed)
   - Client name
   - Due date
4. Click project to view details

### Project Details
Each project shows:
- Project name and number
- Client information
- Project timeline
- Deliverable requirements
- Current status
- Limn contact person
- Special instructions

### Project Status Stages
1. **Assigned** - Project assigned to you
2. **In Progress** - Working on deliverables
3. **Under Review** - Limn team reviewing submissions
4. **Revisions Needed** - Changes requested
5. **Approved** - Design approved
6. **In Production** - Manufacturing stage

## Submitting Deliverables

### Upload Process
1. Go to project details
2. Click "Submit Deliverable"
3. Select deliverable type:
   - Shop drawings
   - 3D models
   - Material specifications
   - Installation guides
   - Other documents
4. Upload file(s)
5. Add version notes
6. Click "Submit"

### File Requirements
- **Formats**: PDF, DWG, SKP, JPG, PNG
- **Max Size**: 50MB per file
- **Naming**: Use descriptive names
- **Versions**: System tracks version history

### Version Control
- All submissions are versioned automatically
- Previous versions remain accessible
- View version history in project details
- Revert to previous version if needed

### Submission Status
Track submission stages:
- **Submitted** - Uploaded successfully
- **Under Review** - Limn team reviewing
- **Approved** - Accepted for production
- **Revisions Needed** - Changes required
- **Rejected** - Does not meet requirements

## Troubleshooting

### Can't Upload File
**Solutions**:
- Check file size (max 50MB)
- Verify file format is supported
- Clear browser cache
- Try different browser
- Contact support if issue persists

### Submission Not Showing
**Solutions**:
- Refresh page
- Check submission status
- Verify upload completed
- Review "Recent Uploads" section
- Contact support with timestamp

### Missing Documents
**Solutions**:
- Check correct project selected
- Verify document type filter
- Ensure you have permissions
- Contact project manager

### Common Questions

**Q: How do I know what deliverables are required?**
A: Check project details page under "Required Deliverables" section.

**Q: Can I submit partial deliverables?**
A: Yes, submit what's ready and note remaining items in version notes.

**Q: How long does review take?**
A: Usually 2-3 business days. Urgent reviews can be expedited.

**Q: Can I update a submission after review starts?**
A: Yes, upload new version. Latest version will be reviewed.

**Q: Who can see my submissions?**
A: Only project team members (Limn staff and client if approved).
