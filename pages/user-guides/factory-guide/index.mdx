# Factory Partner Guide

Complete guide for factory partners using the Limn Systems Factory Portal.

## Getting Started

### Accessing the Factory Portal
1. Navigate to app.limn.us.com/auth/factory
2. Enter your email address
3. Click "Send Magic Link"
4. Check your email and click the link
5. You'll be logged in automatically

### Portal Overview
The Factory Portal enables you to:
- View assigned production orders
- Update production status
- Upload quality reports
- Manage shipping information
- Access production documents

## Dashboard

Your dashboard displays:
- **Active Orders** - Current production orders
- **Pending QC** - Orders awaiting inspection
- **Ready to Ship** - Completed orders
- **Production Metrics** - Your performance stats
- **Urgent Orders** - Priority items

## Managing Production Orders

### Viewing Orders
1. Navigate to Orders
2. View all assigned orders
3. Filter by:
   - Status (pending, in production, complete)
   - Priority (urgent, high, normal)
   - Client
   - Due date
4. Click order to view details

### Order Details
Each order shows:
- Order number and date
- Line items and quantities
- Shop drawings and specs
- Production deadline
- Special instructions
- Limn contact person
- Client details

### Updating Production Status
1. Go to order details
2. Click "Update Status"
3. Select new status:
   - Materials Ordered
   - In Production
   - Assembly Complete
   - Quality Check
   - Ready to Ship
4. Add progress notes
5. Upload photos (optional)
6. Click "Save"

### Production Timeline
Orders progress through:
1. **Order Received** - Initial assignment
2. **Materials Ordered** - Components sourced
3. **In Production** - Manufacturing started
4. **Assembly Complete** - Product finished
5. **Quality Check** - Internal QC done
6. **Ready to Ship** - Awaiting pickup/delivery

## Quality Management

### Uploading QC Reports
1. Navigate to Quality
2. Select order
3. Click "Upload QC Report"
4. Complete QC checklist:
   - Dimensions verified
   - Materials correct
   - Finish quality
   - Hardware installed
   - Packaging adequate
5. Upload inspection photos
6. Add notes
7. Submit report

### QC Photo Requirements
- **Quantity**: Minimum 5 photos per item
- **Views**: Front, back, sides, details, overall
- **Quality**: Clear, well-lit, in-focus
- **Format**: JPG, PNG (max 10MB each)
- **Naming**: Include order number

### QC Issues & Resolutions
If quality issues found:
1. Document issue in QC report
2. Take detailed photos
3. Notify Limn immediately
4. Await instructions
5. Track resolution
6. Re-inspect after fixes

## Shipping Management

### Marking Orders as Shipped
1. Go to order details
2. Click "Mark as Shipped"
3. Enter:
   - Carrier name
   - Tracking number
   - Ship date
   - Number of packages
   - Estimated delivery
4. Upload BOL/packing slip
5. Click "Submit"

### Shipping Documentation
Required documents:
- **Bill of Lading (BOL)** - Carrier receipt
- **Packing List** - Contents inventory
- **Commercial Invoice** - For international
- **Photos** - Packaged items
- **Tracking Info** - Carrier tracking number

### Coordinating Pickups
1. Review shipping requirements
2. Prepare items for pickup
3. Notify Limn 24 hours before ready
4. Ensure proper packaging
5. Have documents ready
6. Update system when picked up

## Document Management

### Uploading Documents
1. Go to order or Documents section

4. Choose file
5. Add description
6. Click "Upload"

## Troubleshooting

### Can't Access Order
**Solutions**:
- Verify order assigned to you
- Check filters not hiding it
- Refresh page
- Contact support with order number

### Photo Upload Failing
**Solutions**:
- Check file size (max 10MB)
- Verify format (JPG/PNG)
- Reduce image resolution
- Try one photo at a time
- Clear browser cache

### Status Update Not Saving
**Solutions**:
- Check required fields filled
- Verify internet connection
- Try again after refresh
- Contact support if persists

### Common Questions

**Q: How do I know when to start an order?**
A: Check order details for "Start By" date. Begin on or before this date.

**Q: What if materials are delayed?**
A: Notify Limn immediately with expected delivery date. They will adjust timeline.

**Q: Can I modify production after starting?**
A: Contact Limn before making changes. Some modifications require approval.

**Q: How many QC photos are required?**
A: Minimum 5 per item: front, back, sides, details, overall view.

**Q: What if I can't meet the deadline?**
A: Notify Limn ASAP (ideally 1 week before deadline). Earlier notice allows for solutions.
