# QC Inspector Guide

Complete guide for QC inspectors using the Limn Systems QC Portal.

## Getting Started

### Accessing the QC Portal
1. Navigate to app.limn.us.com/auth/qc
2. Enter your email address
3. Click "Send Magic Link"
4. Check your email and click the link
5. You'll be logged in automatically

### Portal Overview
The QC Portal allows you to:
- Create and manage inspections
- Upload inspection photos
- Link inspections to orders
- View inspection history
- Update quality status

## Dashboard

Your dashboard displays:
- **Pending Inspections** - Orders awaiting QC
- **Active Inspections** - Currently in progress
- **Completed Today** - Today's inspections
- **Quality Metrics** - Pass/fail rates
- **Urgent Items** - Priority inspections

## Creating Inspections

### Starting a New Inspection
1. Navigate to Inspections
2. Click "Create Inspection"
3. Enter:
   - Order number (or search)
   - Inspector name
   - Inspection date
   - Inspection location
4. Click "Start Inspection"

### Inspection Checklist
For each item, verify:
- ✅ **Dimensions** - Match shop drawings
- ✅ **Materials** - Correct per specifications
- ✅ **Finish Quality** - No defects, smooth
- ✅ **Hardware** - Properly installed
- ✅ **Assembly** - Secure and correct
- ✅ **Packaging** - Adequate protection
- ✅ **Labeling** - Clear and accurate
- ✅ **Documentation** - Complete

### Recording Results
For each checkpoint:
1. Select Pass/Fail/NA
2. Add notes if needed
3. Take photos of any issues
4. Document measurements
5. Note required corrections

### Completing Inspection
1. Review all checkpoints
2. Upload all photos
3. Add overall notes
4. Select final status:
   - ✅ **Passed** - Approved for shipment
   - ⚠️ **Conditional Pass** - Minor fixes needed
   - ❌ **Failed** - Major issues, cannot ship
5. Click "Submit Inspection"

## Photo Management

### Enhanced Photo Upload
The QC Portal features a bulk photo upload system:

#### Uploading Photos
1. Navigate to Upload
2. Click "Select Photos" or drag and drop
3. Upload multiple photos at once
4. System displays thumbnails
5. Link photos to inspections

#### Linking Photos to Inspections
1. After upload, photos appear in grid
2. For each photo:
   - Select from dropdown: "Link to Inspection"
   - Choose order number
   - Choose specific inspection
   - Click "Link"
3. Photo is associated with that inspection

#### Photo Requirements
- **Format**: JPG, PNG
- **Size**: Max 10MB per photo
- **Quality**: Clear, well-lit, in-focus
- **Views**: Multiple angles of each item
- **Quantity**: Minimum 5 photos per item

#### Organizing Photos
- View by inspection
- Search by order number
- Filter by date
- Sort by upload time
- Delete unlinked photos

### Photo Best Practices
- Take photos before and after inspection
- Capture defects close-up
- Include measurement verification
- Show overall and detail views
- Use good lighting
- Keep camera steady

## Managing Inspections

### Viewing Inspection History
1. Navigate to History
2. View all completed inspections
3. Filter by:
   - Date range
   - Order number
   - Pass/fail status
   - Inspector
4. Click inspection to view details

### Inspection Details
Each inspection shows:
- Order information
- Inspection date and time
- Inspector name
- Checklist results
- Photos (linked)
- Overall status
- Notes and comments

### Updating Inspection Status
For conditional passes or failed items:
1. Find inspection in History
2. Click "Update Status"
3. Select new status:
   - Re-inspection scheduled
   - Issues resolved
   - Approved after fixes
   - Still pending
4. Add update notes
5. Upload new photos if needed
6. Submit update

### Re-Inspections
When items need re-inspection:
1. Note issues in original inspection
2. Schedule re-inspection date
3. Create new inspection linked to order
4. Reference original inspection number
5. Verify all issues resolved
6. Approve if satisfactory

## Troubleshooting

### Can't Find Order
**Solutions**:
- Verify order number is correct
- Check order has been assigned
- Search by alternative identifiers
- Contact Limn support

### Photo Upload Failing
**Solutions**:
- Check file size (max 10MB)
- Verify format (JPG/PNG)
- Try uploading fewer photos at once
- Check internet connection
- Clear browser cache

### Inspection Won't Submit
**Solutions**:
- Verify all required fields completed
- Check all checkpoints have status
- Ensure at least one photo uploaded
- Review error messages
- Contact support if issue persists

### Can't Link Photos
**Solutions**:
- Ensure inspection exists
- Verify order number correct
- Check photo uploaded successfully
- Try linking from inspection details page
- Contact support

### Common Questions

**Q: How long should an inspection take?**
A: Average 30-45 minutes per order, depending on complexity and number of items.

**Q: What if I find issues after approving?**
A: Contact QC manager immediately. Create new inspection documenting newly found issues.

**Q: Can I do partial inspections?**
A: Yes, but note which items inspected and which remain. Complete within same day.

**Q: How many photos should I take?**
A: Minimum 5 per item. Take more for complex items or when issues found.

**Q: What if factory disagrees with my findings?**
A: Document thoroughly with photos and measurements. QC manager will review and make final decision.
