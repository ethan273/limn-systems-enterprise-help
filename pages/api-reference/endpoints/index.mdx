## Core Business Logic

### CRM & Sales (18 endpoints)

Manage contacts, leads, customers, and sales pipeline.

**Endpoints:**
- `crm.contacts.list` - List all contacts with filters
- `crm.contacts.getById` - Get single contact by ID
- `crm.contacts.create` - Create new contact
- `crm.contacts.update` - Update contact details
- `crm.contacts.delete` - Delete contact
- `crm.contacts.convertToLead` - Convert contact to lead
- `crm.leads.list` - List all leads
- `crm.leads.getById` - Get lead details
- `crm.leads.convertToProspect` - Convert lead to prospect
- `crm.prospects.list` - List all prospects
- `crm.prospects.getById` - Get prospect details
- `crm.prospects.convertToClient` - Convert prospect to client
- `crm.customers.list` - List all customers
- `crm.customers.getById` - Get customer details
- `crm.activities.list` - List CRM activities
- `crm.activities.create` - Log CRM activity
- `crm.pipeline.get` - Get sales pipeline view
- `crm.analytics.get` - Get CRM analytics
