### Frontend
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- tRPC v10
- Prisma ORM
- PostgreSQL
- Supabase Auth

### Infrastructure
- Vercel (hosting)
- Supabase (database, auth, storage)
- Upstash Redis (caching, rate limiting)
- Resend (email)
